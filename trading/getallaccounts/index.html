    Get All Accounts

    How to get a list of all accounts Overview

    This article will show you how to use the API to get a list of accounts. You will learn how to: - Listen for account changes - Subscribe to account changes - Get and print accounts - Print account properties

    You will need some basic knowledge of TypeScript and JavaScript to follow this tutorial.
    You can also download the sample code at the end of the article.

    Step 1: Listen for account changes

    The first step is to create an account change listener.
    This is a function that will be called whenever an account is added, removed, or updated.
    You can use the AccountInfo interface to access the account details.

    Below you can see some example code to create an account change listener:

    class AccountChangeListener implements FXConnectLite.IAccountChangeListener {
    onChange(accountInfo: FXConnectLite.AccountInfo): void {
    }
    onAdd(accountInfo: FXConnectLite.AccountInfo): void {
    }
    onRefresh(): void {
    }
    onDelete(accountInfo: FXConnectLite.AccountInfo): void {
    }
}

    let AccountChangeListener = /** @class */ (function () {
    function AccountChangeListener() {
    }
    AccountChangeListener.prototype.onChange = function (accountInfo) {
    //some code on change
    };
    AccountChangeListener.prototype.onAdd = function (accountInfo) {
    //some code on add
    };
    AccountChangeListener.prototype.onRefresh = function () {
    //some code on refresh
    };
    AccountChangeListener.prototype.onDelete = function (accountInfo) {
    //some code on delete
    };
    return AccountChangeListener;
}());

    For more information about the AccountInfo interface, see the API reference.

    Step 2: Subscribe to account changes

    The next step is to get an instance of the account manager and subscribe to account changes. The account manager is an object that provides methods for managing accounts. You can use the IAccountsManager interface to access its methods.

    To get an instance of the account manager and subscribe to account changes, you need to write some TypeScript or JavaScript code like this:

    let accountChangeListener = new AccountChangeListener(); // create account change listener
session.getAccountsManager().subscribeAccountChange(accountChangeListener); // subscribe

    let accountChangeListener = new AccountChangeListener(); // create account change listener
session.getAccountsManager().subscribeAccountChange(accountChangeListener); // subscribe

    For more information about the IAccountsManager interface, see the API reference.

    Step 3: Get and print accounts

    The third step is to create a function that will get and print the list of accounts. You can use the getAccountsSnapshot method of the account manager to get a snapshot of the current accounts. This method returns a promise that resolves with an array of AccountInfo objects.

    To create a function that will get and print the list of accounts, you need to write some TypeScript or JavaScript code like this:

    private getAndPrintAccounts = async (session, application) => {
    return new Promise<void>((resolve, reject) => {
        application.resolveGetAccounts = resolve;
        let accountsManager = session.getAccountsManager();
        application.printAccountsSnapshot(accountsManager);
    })
}

    this.getAndPrintAccounts = function (session, application) {
    return new Promise((resolve, reject) => {
        application.resolveGetAccounts = resolve;
        var accountsManager = session.getAccountsManager(); // get account manager
        accountsManager.getAccountsSnapshot(this.getAccountsSnapshotCallback); // get account snapshot
    });
};

    For more information about the getAccountsSnapshot method, see the API reference.

    And the code of callback for providing information about all accounts:

    The code of callback

    private getAccountsSnapshotCallback = new class implements FXConnectLite.IGetAccountsSnapshotCallback {
    onAccountsReceived(accounts: FXConnectLite.Account[]): void {
        accounts.forEach(account => {
            Printer.print(`Current snapshot:`);
            Application.printAccount(account);
        });
        application.resolveGetAccounts();
    }
}();

    this.getAccountsSnapshotCallback = new /** @class */ (function () {
    function snapShotCallback() {
    }
    snapShotCallback.prototype.onAccountsReceived = function (accounts) {
        accounts.forEach(function (account) {
            console.log("Current snapshot:");
            Application.printAccount(account); // for each account show info
        });
        application.resolveGetAccounts();
    };
    return snapShotCallback;
}())();

    See also onAccountReceived
