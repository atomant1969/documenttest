Download and Install

To download and install FCLite API, you need to follow these steps:

Go to the official website of Gehtsoft USA LLC, the software development company that created FCLite API for Forex Capital Markets (FXCM): https://gehtsoftusa.com/ Click on the Products tab and select FCLite from the drop-down menu. You will be redirected to the FCLite documentation page: https://docs.gehtsoftusa.com/fclite/#howto.html Choose the programming language that you want to use with FCLite API: C++, Java, .NET, or Python. You will see the instructions on how to download and install the corresponding FCLite library for your language. Follow the instructions carefully and make sure that you have all the required dependencies and tools for your language and platform. For example, if you want to use FCLite with C++, you will need a C++ compiler that supports C++11 standard, such as Visual Studio 2015 or later, GCC 4.8 or later, or Clang 3.3 or later. After you download and install the FCLite library for your language, you can start using it in your application. You will need to include the FCLite header files or import the FCLite modules in your code, depending on your language. You will also need to link your application with the FCLite library files or reference the FCLite assemblies in your project, depending on your platform. To test if your installation is successful, you can try to run some of the examples provided by Gehtsoft USA LLC for each language. You can find the examples here: https://docs.gehtsoftusa.com/fclite/#examples.html If you encounter any problems or have any questions during the installation process, you can contact Gehtsoft USA LLC's support team at support@gehtsoftusa.com or visit their website at https://gehtsoftusa.com/.
